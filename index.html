Study Projects

These projects were created as part of a module or class of my course.

Library Project

The Library project was an agile study project using Scrum, GitHub, Java and JavaFX.

It was created to be a management software for a fictitious library and allows managing books and book returns and borrows.

Flight Radar

We developed the Flight Radar as part of a lab. It utilizes Java and JavaFX, version control was done using GitLab.

It uses a publicly available online API and allows the tracking of aircrafts in a 150km range.

Minesweeper

The Minesweeper project was also part of a lab. We used JavaScript to create the whole page and for the game logic and CSS for the styling.

It features local and remote logic - the game can either be handled locally by JavaScript or remotely by a server set up by the lecturer.
